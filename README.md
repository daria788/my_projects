# Cup Russia 2026 — Запуск бренда AITHER
**Задача:** выбрать перспективную нишу для запуска нового бренда на маркетплейсах, обосновать выбор и разработать стратегию вывода с финансовой моделью.

## Что сделано:
>- Выступила в роли Project Lead и аналитика — координировала команду и готовила финальное решение
>- Проанализировала рынки Wildberries и Ozon в категории «Красота и здоровье» (объём 1 039 млрд ₽)
>- Обосновала выбор ниши — парфюмированные гели для душа (объём 47,3 млрд ₽, рост +43,7%, 8 из 20 топ-позиций на WB)
>- Разработала стратегию позиционирования и ценообразования для бренда AITHER (399–449 ₽ за единицу, 799 ₽ за набор)
>- Построила финансовую модель с прогнозом выручки на 3 года (целевая маржинальность 31%, окупаемость к началу 3-го года)
>- Проверила 4 гипотезы: выбор категории, драйвер покупки (эмоция vs функция), канал продаж (маркетплейсы vs D2C), мультиканальность

**Результат:** Разработана стратегия, позволяющая войти в топ-3 с прогнозом выручки и остатком ДП 229 млн ₽ к началу 3-го года.
